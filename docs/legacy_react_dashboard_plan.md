### 핵심 기능 스펙

#### (A) Base 이미지 업로드
- 드래그&드롭 또는 클릭
- 지원 형식: PNG (투명 배경), WebP

#### (B) Face 그리드 오토 슬라이싱
- 업로드 규격: **2400 × 3240 px** 템플릿 1장
- 브라우저 Canvas로 **800 × 1080** × 9칸 자동 분할 (서버 불필요)
- 슬롯별 태그 지정:

```
┌────────┬────────┬────────┐
│ #Neutral│ #Happy │ #Sad   │  1행
├────────┼────────┼────────┤
│ #Angry  │#Surprise│#Embarrass│ 2행
├────────┼────────┼────────┤
│ #Scared │#Thinking│#Custom │  3행
└────────┴────────┴────────┘
   800px    800px    800px
```

#### (C) 앵커(Anchor) 좌표 지정
- Base 이미지 위 캔버스에서 **마우스 클릭** → 얼굴 중심축 (X%, Y%) 저장
- 십자선 마커로 지정 위치 시각화
- 뷰어에서 Face 파츠가 이 좌표 기준으로 position: absolute 합성

#### (D) 이모지 ↔ 태그 매핑 편집
- 대화 텍스트 내 이모지 자동 감지 → 표정 자동 전환
- 사용자 정의 매핑 추가/수정 가능

### 상세 컴포넌트 설계

#### [NEW] `apps/web/app/dashboard/character/page.tsx`

## 창 ④ — 비주얼 노벨 뷰어 (`/session/[id]`)

> 실제 TRPG 세션 진행 화면. 몰입형 비주얼 노벨 UI.

### Z-Index 레이어 구조

```
Layer 5 (z:500) : 사전 툴팁 팝업
Layer 4 (z:400) : 토스트 시스템 알림
Layer 3 (z:300) : 대사창 (중앙 세로형 블러 박스)
Layer 2 (z:200) : 발언 중인 캐릭터 (팝업 애니메이션)
Layer 1 (z:100) : 대기 중 캐릭터 (반투명)
Layer 0 (z:0)   : 배경 이미지 (크로스페이드)
```

### 화면 전체 레이아웃

```
┌──────────────────────────────────────────────────────────────┐
│  [배경 이미지 — 전체 화면, 크로스페이드]                     │
│                                                              │
│                 ╔═══════════════════════╗                    │
│  [캐릭 대기]   ║   < 대사창 블러 박스 > ║  [캐릭 대기]    │
│  opacity:0.5   ║   left:15% right:15%  ║  opacity:0.5     │
│  z-index:100   ║   top:0 ~ bottom:0    ║  z-index:100     │
│                 ║   테두리 없음          ║                   │
│  [발언 캐릭]   ║   가장자리 페이드아웃  ║                   │
│  scale(1.05)   ║   (mask gradient)     ║                   │
│  z-index:200   ║                       ║                   │
│  translateY    ║   발언자:             ║                   │
│  (-20px)       ║     대화 텍스트 출력   ║                   │
│                 ║     대화 텍스트 ▶     ║                   │
│                 ╚═══════════════════════╝                    │
│                                                              │
│  🟢 🎙 ▶ 1.0x ⛶  [컨트롤 바 — 좌상단]                      │
└──────────────────────────────────────────────────────────────┘
```

### 대사창 (DialogueBox) 스펙

| 속성 | 값 |
|------|-----|
| 가로 위치 | left: 15% ~ right: 15% (화면의 70%) |
| 세로 위치 | top: 0 ~ bottom: 0 (화면 전체 높이) |
| 배경 | rgba(0,0,0,0.75) — 디스코드 같은 어두운 회색 계열 (블러보다 검게) |
| 포인트 컬러 | #949BA4 등 디스코드 무채색/회색 톤 |
| 블러 | backdrop-filter: blur(2px) — 최소 |
| 테두리 | 없음 (border: none) |
| 페이드 | 상하좌우 mask-image gradient (자연스럽게 소멸) |
| 텍스트 레이어 | 마스크 밖 별도 레이어 (항상 선명) |
| 텍스트 위치 | padding: 20vh 15% (위아래 + 좌우 여백) |

#### 텍스트 구조

```
발언자:
    대화 텍스트가 한 글자씩 타이핑되며 출력됩니다■
    완료 시 → 텍스트 끝에 인라인으로 ▶ 표시 (오른쪽 이동 애니메이션)
```

- `발언자` : Cinzel 폰트, gold 색상, 좌측 정렬, `::after { content: ":" }` 자동 추가
- 텍스트 : `padding-left: 2rem` 들여쓰기
- `▶` : `display: inline`, 오른쪽 움직이는 pulse 애니메이션

| 상태 | opacity | filter | z-index | transform |

#### 파츠 합성

```

→ anchorX%, anchorY% 기준: left: anchorX%, top: anchorY%,
   transform: translate(-50%, -50%)
```
